# 💬 Real-Time Chat App

A modern **real-time chat application** built for instant messaging with a smooth and responsive user experience.
Supports live communication, multiple users, and dynamic updates without page refresh.

---

## 🚀 Features

* ⚡ Real-time messaging (no refresh required)
* 👤 User join/leave notifications
* 💬 One-to-one and group chat support
* 🔔 Instant message updates
* 🌐 Responsive design (mobile + desktop)
* 🌙 Light / Dark mode UI
* 📡 WebSocket-based communication
* 🔒 Secure message handling

---

## 🛠️ Tech Stack

### Frontend

* HTML5, CSS3, JavaScript
* Responsive UI with modern styling

### Backend

* Python (Flask / FastAPI) OR Node.js (Socket.io)
* WebSockets for real-time communication

### Database (optional)

* SQLite / PostgreSQL / MongoDB

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/bidishajoshi/realtime-chat-app.git
cd realtime-chat-app
```

### 2. Create virtual environment

```
python -m venv venv
venv\Scripts\activate   # Windows
```

### 3. Install dependencies

```
pip install -r requirements.txt
```

### 4. Run the app

```
python app.py
```

---

## 🌐 Usage

* Open browser: `http://localhost:5000`
* Enter username
* Start chatting in real-time 🚀
